# User Management CRUD App

A simple User Management CRUD project built using **HTML, CSS, and Vanilla JavaScript**.

This project was created to practice JavaScript fundamentals, DOM manipulation, event handling, and dynamic UI rendering without using any frameworks or libraries.

## 🚀 Features

* Add users dynamically using a form
* Display user information as cards
* Delete users from the UI
* Validate form inputs to prevent empty submissions
* Generate unique IDs for each user
* Automatically reset the form after successful submission
* Re-render the UI after every state change

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (Vanilla JS)

## 📚 JavaScript Concepts Practiced

* DOM Selection (`getElementById`)
* Event Handling (`addEventListener`)
* `preventDefault()`
* Reading Input Values (`.value`)
* Arrays and Objects
* Dynamic UI Rendering using `forEach()`
* Template Literals
* Refactoring repeated logic into reusable functions
* Inline Events (`onclick`)
* Function Arguments
* `findIndex()`
* `splice()`
* Form Validation using `trim()`
* Unique ID generation using `Date.now()`
* Form Reset using `form.reset()`

## 💡 Key Learning

One of the biggest lessons from this project was understanding that programming is an iterative process.

Build something → Identify problems → Understand why they happen → Refactor and improve the solution.

Through this project, I learned the importance of writing reusable code and maintaining a single source of truth using arrays and rendering functions.
